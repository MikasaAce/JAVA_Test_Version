## 概述

本标准规定了C/C++语言严于语法规定的编程安全子集，适用于C/C++语言编写的军用安全关键软件。标准中的准则分为：

- **强制准则（Required Rule）**：软件编程中强制要求遵循的编码准则
- **建议准则（Advisory Rule）**：软件编程中推荐参照执行的编码准则
